# GitStart ## Markdown 문법
1.**헤더(Headers)**:

'''
  벡틱(') <-- 3개 코드라인 markdown 문법이 적용안됨
  #헤더1
 ##헤더2
###헤더3
'''

2. 굵게
'''**굵게**'''
**굵게**

3.기울임
'''*기울임*'''
*기울임*

4.링크
'''[링크텍스트](url)'''
[링크텍스트](url)

5.이미지
'''![대체 텍스트](이미지_url)'''
![대체 텍스트](이미지_url)

6. 리스트
'''1. 번째 2. 번째'''
1. 번째 2. 번째


7.인용
